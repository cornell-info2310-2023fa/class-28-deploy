# INFO 2310

Open this repository as a codespace.

## Client

```sh
cd client
```

**Install dependencies:**

```sh
npm install
```

**Start development server:**

```sh
npm start
```

## Server

```sh
cd server
```

**Install dependencies:**

```sh
npm install
```

**Initialize database:**

```sh
mongosh init.mongo.js
```

**Start development server:**

```sh
npm run dev
```
